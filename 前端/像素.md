#### 一、窗口平铺（方案一）

```html
<script>
      document.documentElement.style.fontSize=document.documentElement.clientWidth/750*100 +'px'
</script>

<style>
body{
  font-size: 16px;
}

.box{
  width: 7.5rem;
  height: 200px;
  background: yellow;
}
</style>

```

#### 二、窗口平铺（方案二）

```html
<script>
    document.documentElement.style.fontSize=document.documentElement.clientWidth/750*16 +'px'

<style>

  height: 12.5rem;
  background: yellow;
}
</style>

```
